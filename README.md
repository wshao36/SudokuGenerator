# SudokuGenerator
Student: Wilson 

Description:
A Java Console Program that generates a randomly solved 9x9 Sudoku board that the user can play.


How to Run?
Compile: javac SudokuGenerator.java
Run: java SudokuGenerator

Generation:
Uses a shift pattern algorithm with the formula shift = (i % 3) * 3 + (i / 3) and fills the array either by row or column
